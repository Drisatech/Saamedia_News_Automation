SaaMedia News Automation API Docs

This API allows scraping, viewing, and managing automated news articles for the SaaMedia Platform.

GET /dashboard

Displays the list of stored articles with search and filter support.

  • ?q=keyword – filter by title/content
  • ?category=Politics – filter by category

GET /run-news

Runs the news agent to scrape and store the latest articles from supported sources.

Example Response:

{
  "status": "running",
  "results": ["Article 1", "Article 2", ...]
}

GET /docs

Displays this documentation page.
